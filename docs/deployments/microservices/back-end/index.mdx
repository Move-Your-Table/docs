# Introduction
Welcome to MYT's back-end service. This is the service directly connected to the main database.
This service is responsible for fetching data and delivering to the Gateway, on it's request.

Under the hood, it runs on a framework called **Nest.JS**, which is a Node.JS framework that is built from the ground up to use **TypeScript**.
