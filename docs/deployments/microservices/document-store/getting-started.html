    $ git git@git.ti.howest.be:TI/2021-2022/s5/trending-topics/projects/hybrid-work1/back-end.git

  2. Define the .env file as detailed in the .env configuration section.

  3. Within the root of the project, run the following command:

    ```bash
    # If you want a development database
    docker-compose up mongodb_dev

    # If you want a production database
    docker-compose up mongodb_prod
    ```
