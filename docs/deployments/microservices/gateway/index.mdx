# Introduction
Welcome to the Gateway service of MYT-MSF. The gateway service is probably the one you'll be interaction the most with.
This service allows you to interact with MYT-MSF, fetch data & perform actions.

The framework is build upon **TS.ed**, a TypeScript framework with a very heavy focus on modularity & self-documentation.
For example, the API spec is self-generated without any configuration or annotations.
