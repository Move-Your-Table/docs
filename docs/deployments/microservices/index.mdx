# What?

**MYT-MSF** (a.k.a Move Your Table's Micro-Services Framework) is a framework consisting of multiple services that power the MYT application.
You're main endpoint will be the **[Gateway](./gateway/index)** service, that provides access to MYT's functionality.
This gateway is connected to other services, such as:
- [Back-end Service](./back-end/index)
- [Automate Service](./automate-server/index)
- [Message Broker Service](./message-broker)
- [Document Store Service](document-store/index)
