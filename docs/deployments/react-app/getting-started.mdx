# Getting started

:::tip codebase
You can find the codebase [here](https://git.ti.howest.be/TI/2021-2022/s5/trending-topics/projects/hybrid-work1/front-end)
:::

1. Clone the code with `git clone`
2. Run Yarn install
   ```bash
   $ npm install -g yarn
   $ yarn install
   ```
3. Copy the `.env.example` file, rename it to `.env` and complete using the [`.env` reference](#env-configuration)
4. Start a development server
   ```bash
   $ yarn start
   ```

> ✅ **Done!** You can reach the application at [localhost:3000](http://localhost:3000)
