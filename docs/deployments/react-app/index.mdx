# Introduction
Welcome to the **MYT Web**. 

MYT Web is built with
- **React**, a JavaScript framework focused on components, modularity & reusability
- **TailwindCSS**, a CSS framework focused on functional CSS classes.

It has 2 parts, employee & admin, which each they own unique features and functionality.
